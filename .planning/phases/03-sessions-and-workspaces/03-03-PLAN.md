  <action>
**1. Update mobile/hooks/useWorkspaces.ts** to add mutation hooks:
- `useCreateWorkspace()`: mutation calling client.createWorkspace(). On success invalidates ['workspaces'].
- `useUpdateWorkspace()`: mutation calling client.updateWorkspace(). On success invalidates ['workspaces'].
- `useDeleteWorkspace()`: mutation calling client.deleteWorkspace(). On success invalidates ['workspaces'].
- `useReorderWorkspaces()`: mutation calling client.reorderWorkspaces(). On success invalidates ['workspaces'].
Keep existing useWorkspaces() and useGroups() hooks unchanged.

**2. Create mobile/components/workspaces/WorkspaceItem.tsx:**

Memoized (React.memo) pressable card for workspace list. Props: `{ workspace: Workspace; sessionCount: number; onPress: (id: string) => void; onLongPress: (id: string) => void; isDragging?: boolean }`.

Layout:
- Left: Color circle (12px, workspace.color or default accent)
- Center column: Workspace name (bold), description (secondary color, numberOfLines={1}), "X sessions" count
- Right: Drag handle icon (three horizontal lines) for reorder mode

When isDragging is true, apply slight scale-up (1.03) and elevated shadow via Reanimated. Use Card from ui components as wrapper. All colors from useTheme().

**3. Create mobile/components/workspaces/WorkspaceActions.tsx:**

ActionSheet wrapper for workspace operations. Props: `{ workspace: Workspace; visible: boolean; onClose: () => void }`.

Actions:
- Rename: ModalSheet with Input, calls useUpdateWorkspace with { name }
- Change Color: ModalSheet with a grid of 14 color swatches (Catppuccin palette colors: rosewater, flamingo, pink, mauve, red, maroon, peach, yellow, green, teal, sky, sapphire, blue, lavender). Tap a swatch calls useUpdateWorkspace with { color: selectedHex }
- Edit Description: ModalSheet with multiline Input, calls useUpdateWorkspace with { description }
- Delete (destructive): Confirm dialog, calls useDeleteWorkspace

Show Toast on success/error. All colors from useTheme().

**4. Create mobile/components/workspaces/NewWorkspaceModal.tsx:**

ModalSheet-based form. Props: `{ visible: boolean; onClose: () => void }`.

Form fields:
- Name (Input, required, placeholder "Workspace name")
- Description (Input, multiline, optional)
- Color picker: Same 14-swatch grid as WorkspaceActions. Default to first accent color.

Submit calls useCreateWorkspace mutation. On success: close modal, show Toast. Validate name is non-empty before submit.
