**1. Create mobile/hooks/useSearch.ts:**

A hook that provides debounced search with two modes: keyword and AI semantic.

```typescript
export function useSearch(options?: { mode?: 'keyword' | 'ai'; workspaceId?: string }) {
  // Internal state: debouncedQuery (300ms debounce), searchMode
  // Returns: { query, setQuery, results, isLoading, isError, searchMode, setSearchMode }
}
```

Implementation:
- Track `query` string in useState
- Debounce it by 300ms into `debouncedQuery` (use useEffect + setTimeout, or a simple useDebounce helper)
- TanStack Query: key ['search', mode, debouncedQuery, workspaceId]. Enabled only when debouncedQuery.length >= 2.
- For keyword mode: call client.search(debouncedQuery, 20)
- For AI mode: call client.searchConversations({ query: debouncedQuery, workspaceId })
- Use `placeholderData: keepPreviousData` (TanStack Query v5 equivalent of keepPreviousData) so previous results stay visible while new results load
- Return the query, setter, results, loading state, error state, and mode controls
